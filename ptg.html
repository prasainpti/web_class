Steps for creating new repository on the command line:

  1. git add init
  2. git add ReadME.md
  3. git commit -m "first commit"
  4. git branch -M main
  5. git remote add origin git@github.com:rahulho/repo_name.git
  6. git push -u origin main

Steps for pushing existing repository from the command line

  • git remote add origin git@github.com:rahulho/repo_name.git
  • git branch -M main
  • git push -u origin main
